##  Фаза 12: Полировка и контент (шаги 331–370)

> Цель: финальный геймплейный срез — всё работает, сбалансировано, играбельно.

### UI Polish (шаги 331–340)

331. Главное меню: Start, Load, Options, Credits (имитация старой игры)
332. Game Over: Restart → return to overworld с потерей половины золота
333. HUD переработан: HP/MP/XP/Hunger полоски, уровень, класс
334. Все окна имеют единый стиль (цвета, шрифты, рамки под ретро)
335. Анимация атаки: биллборд взмаха оружия (2 кадра)
336. Анимация заклинания: биллборд эффекта на цель (искры, огонь, лёд)
337. Анимация получения урона: красная вспышка на HUD
338. Анимация смерти монстра: биллборд исчезает с частицами
339. Перетаскивание предметов: drag-and-drop между инвентарём и экипировкой
340. Адаптивный UI: retro режим (фиксированное разрешение) и native (масштабируемое)

### Saving Expansion (шаги 341–346)

341. Переписать `SaveGame` на сохранение полного состояния в JSON
342. Сохраняется: класс, уровень, XP, статы, hunger, инвентарь, экипировка
343. Сохраняется: изученные заклинания/умения, activeBuffs, questLog
344. Сохраняется: текущая локация (overworld или dungeon + этаж + позиция)
345. Автосохранение: при входе в подземелье, выходе из него, после важных квестов
346. Слоты сохранения: 3 слота, выбор при загрузке

### Balance Pass (шаги 347–356)

347. Все монстры проверены: урон соответствует уровню, XP адекватна
348. Все предметы проверены: статы соответствуют материалу и редкости
349. Cost таблица для крафта: ингредиенты не дороже готового предмета
350. Hunger: скорость расхода не душит игрока, еды хватает
351. Мана: регенерация позволяет кастовать 1-2 заклинания за бой
352. Боссы: сложнее обычных монстров ×2 HP, уникальные механики
353. Кривая сложности: плавный рост, без резких скачков
354. Экономика: золото не сыплется, но хватает на базовое снаряжение
355. Рецепты: ингредиенты доступны на соответствующих этажах
356. Лут: каждый этаж даёт материалы для крафта следующего уровня

### Bug Fixing (шаги 357–363)

357. Проверка всех JSON на валидность при загрузке
358. Все переходы между GameState: нет утечек памяти
359. Проверка OpenGL: нет утечек текстур/шейдеров/материалов
360. Edge cases: пустой инвентарь, 0 HP, 0 MP, уровень 1, максимальный уровень
361. Сборка с флагами -Wall -Wextra -Werror: 0 предупреждений
362. Тест под MSVC: 0 ошибок
363. Тест: 30 минут game time без крешей

### Sound & Music (шаги 364–370)

364. `data/sounds.json` — звуковые эффекты (step, hit, die, spell, pickup, door, trap)
365. Загрузка .ogg файлов звуков через `AudioSystem`
366. Звук шага: при каждом движении на 1 клетку (overworld + dungeon)
367. Звук атаки: при успешной атаке, крите, промахе
368. Звук заклинания: по типу элемента (fire, ice, lightning, holy)
369. Фоновая музыка: одна тема для overworld, одна для dungeon, одна для боя
370. Настройка громкости в Options (F2) — работает через `AudioSystem::SetChannelVolume`
